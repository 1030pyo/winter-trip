# Winter Trip — 홋카이도 겨울 여행 플래너

홋카이도 겨울 여행 코스를 짜는 인터랙티브 웹 페이지. 일행 구성 → 지도에서 행선지(전광판) 루트 짜기 → 설문 → 맞춤 플랜까지 한 파일로 동작하는 정적 사이트야.

## 실행
별도 빌드 없이 `winter.html`을 브라우저로 열면 돼. (이미지·폰트·스프라이트 등 같은 폴더의 에셋을 상대경로로 불러옴)

## 구성
- `winter.html` — 페이지 전체(HTML/CSS/JS 인라인)
- `Asset/`, `sprites/`, `Font/`, `Palette/`, `Examples/` — 이미지·폰트·팔레트·참고 에셋

## 배포
GitHub Pages로 호스팅. 저장소가 곧 백업이며, 커밋 기록으로 버전 관리가 됨.
